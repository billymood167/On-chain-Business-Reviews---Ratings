# On-chain Business Reviews & Ratings
A decentralized business review system built on Stacks blockchain.

## 🌟 Features

- Verified customer reviews
- DAO-managed review moderation
- Purchase verification
- Automated rating calculations
- Secure review flagging system

## 🛠️ Technical Details

- Smart contract written in Clarity
- Comprehensive test suite
- Security features:
  - Purchase verification
  - DAO access control
  - Input validation
  - Review authenticity checks

## 🚀 Getting Started

1. Install Clarinet
```bash
curl -L https://github.com/hirosystems/clarinet/releases/download/v1.0.0/clarinet-linux-x64.tar.gz | tar xz
```

2. Initialize project
```bash
clarinet new on-chain-reviews
```

3. Run tests
```bash
clarinet test
```
